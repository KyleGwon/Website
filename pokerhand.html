About the program

Completion Date: October 25, 2018

Program Functionality

This program uses an existing set of data to determine what the user most likely has when given two cards.

Planning

For this project, I used Bayes Theorem. It was very complicated for me, having only just starting Algebra at the time. However, I had someone to teach me in person, which helped a lot. I had decided to use many functions, as opposed to doing it all in one or two. This would make it easier to focus on one aspect at a time, and make less mistakes. This would potentially make the file easier to read, as well.

Functions

readwriteCSV()
This function can read or write to a file specified.

sorter()
Gets raw data from file specified. Returns the information each hand and what each card hand is as two lists.

suitProb()
Calculates the probability of getting a certain hand (pair, flush, etc. using the suit.

suit()
Finds the suit of each card, then uses suitProb() to find the probability of each suit being in a certain hand.

rankProb()
Calculates the probability of getting a certain hand (straight, full house) using the rank.

rank()
Finds the rank of each card, then uses rankProb() to find the probability of each rank being in a certain hand.

vectorProbs()
Calculates the probabilties of each hand (three of a kind, straight flush, etc.)

prob()
Finds the total probability of being a certain hand.

normalize()
Just makes the list neater.

What I Learned

The main thing that I learned was how to utilize Bayes Theorem. Bayes Theorem uses an existing set of data to predict the outcome based on a new information. In this project, I had to use the cards the user inputed as new information, and the possible five-card hands as outcomes.
